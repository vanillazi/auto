开发者

思路​

应用程序通过媒体投影实现屏幕截图,采用OpenCV对屏幕截图执行图像处理.手势操作由无障碍服务执行,以便执行各种操作,例如点击,滑动或滚动屏幕.

开发环境​

  • Windows 11 Pro 10.0.22621
  • OpenJDK 11
  • Android Studio Electric Eel | 2022.1.1 Patch
  • Android SDK 32.
  • Android NDK,
  • Cmake

使用​

安装​

用户下载源代码,构建APK,将建构好的APK发送到手机进行安装,或通过USB直接安装.

注意事项:因程序使用的媒体投影,无障碍服务都是高危敏感操作,强烈建议用户自行下载代码编译,切勿轻信任何第三方提供的本程序安装包

参考​

  2. 无障碍服务
  3. OpenCV
