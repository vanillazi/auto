使用自动助手实现小窝钓鱼

小窝钓鱼流程分析​

小窝钓鱼分为四个状态

  • 空闲
  • 钓鱼中
  • 钓到鱼
  • 炫耀鱼

如下图所示:

空闲状态判断​

当玩家处于空闲时,鱼竿的操作图标如下如所示

我们一旦检测到这个图标,可以认为玩家处于空闲状态.

钓鱼中状态判断​

当玩家处于钓鱼中时,鱼竿的操作图标与空闲状态保持一致,我们无法通过操作图标判断,暂时也没有其它合适的方法进行判断.我们可以设定玩家处于空闲状态时,程序点击鱼竿的操作图标后的一段时间内,玩家的状态处于钓鱼中,超过一定时间或者鱼上钩后,切换到其他状态.

钓到鱼状态判断​

当玩家钓到鱼时,鱼竿的操作图标如下如所示

我们一旦检测到这个图标,可以认为玩家已经钓到鱼了.

炫耀鱼状态判断​

当玩家处于炫耀鱼状态时,鱼竿的操作图标与空闲状态保持一致,我们无法通过操作图标判断,暂时也没有其它合适的方法进行判断. 我们可以设定玩家钓到鱼后的一段时间内,玩家处于炫耀鱼的状态,超过一定时间后,玩家自动切换到空闲状态.

备注: 炫耀鱼的时长根据鱼大小不同而不同,最长为6250毫秒.

自动流程设计​
