免责声明​

这是一个用Java编写的Android应用程序,用于教育研究目的,用于研究Android应用开发架构,并将媒体投影和无障碍服务等系统服务与OpenCV结合使用.通过图像模板匹配自动执行某些工作流程.

下载此应用程序，即表示你同意你的帐户可能因过度使用而被标记并被封号。我对你使用此应用程序的程度不承担任何责任。我相信你有必要的自我控制，在合理的时间内使用此应用程序，并在两者之间给自己休息。

因为运行环境存在差异,本应用程序所提供的各项功能并不能保证在任何情况下都能正常或者如期执行.用户使用本应用程序所产生的一切后果,我不承担任何责任。

终端要求​

  • 手机运行内存8G及以上.
  • 不少于1G的可用存储空间.
  • 支持Ipv6网络.
  • Android系统版本9.0及以上.推荐Android系统版本11.0及以上.

安装​

用户下载源代码,构建APK,将建构好的APK拷贝到手机或者通过USB进行安装.

注意事项:因程序使用的媒体投影,无障碍服务都是高危敏感操作,强烈建议用户自行下载代码编译,切勿轻信任何其他第三方提供的本程序安装包
