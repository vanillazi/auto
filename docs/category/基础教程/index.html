基础教程

全面了解自动助手的基本功能

📄️ 概览

本教程将简单介绍自动助手的基本功能,包括基础服务,对象,步骤,流程,流程实例,设置.

📄️ 基础服务

基础服务由屏幕录像,自动点击,自动输入三部分组成,是实现自动操作的基础.

📄️ 对象

对象是对用户界面元素的一个统一的抽象,包括图标,文本,像素,点,矩形,控件.用户通过自定义对象后,才可以在对象上执行一定的操作.对象分为一般对象和可匹配对象.

📄️ 步骤

步骤由执行条件,执行动作,结束条件组成,是用户在某场景下操作手机的抽象.

📄️ 流程

流程是指一系列的操作步骤,按照用户需求的有序组合.

📄️ 流程实例

用户定义并正在执行的流程称为流程实例.

📄️ 配置

介绍自动助手的常用配置,授权配置,调试日志,关于等信息的查看.

📄️ 祝贺你

你对自动助手的基本概念已经有了一个全面的了解,开始你的自动化工作吧!
